Command line Command
====================
Default      - mvn spring-boot:run -Dspring.profiles.active
Development  - mvn spring-boot:run -Dspring.profiles.active=dev
Production   - mvn spring-boot:run -Dspring.profiles.active=prod

Edit Application Configuration Command
======================================
-Dspring.profiles.active=dev

Call another way to application.properties
=========================================
spring.profiles.active=dev
